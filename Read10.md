# Error Handling & Debugging

## Type Error Messages

- Error
- SyntaxError
- ReferenceError
- TypeError
- RangeError
- URIError
- EvalError

The JavaScript console log will tell you the what type of error it is and
helps you find the bug. When debugging it could help to look at error
in more than one browser.
